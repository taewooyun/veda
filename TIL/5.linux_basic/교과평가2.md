### Startup code
    - .global _start

    ```
    _start:
            mov r1, #10       ; count = 10 EA
            mov r2, #0        ; sum
    L1:
            add r2, r2, r1    ; r2 = r2 + r1
            subs r1, r1, #1    ; 10 -> 9 -> 8 ... -> 0 // ALU -report -> Flag Z=1 
            ; report를 하려면 명령어에 s를 붙여야함
            bNE L1

            mov r0, r2        ; r0 <- r2
    ```

    해석을 해야함
