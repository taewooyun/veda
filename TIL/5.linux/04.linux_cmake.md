# cmake
빌드 시스템을 만들어주는 시스템

make에서 include config의 역할

```bash
$ cmake [options] <path-to-source>
$ cmake [options] <path-to-existing-build>

$ cmake [options] -S <path-to-source> -B <path-to-build>
$ -G : Generate, makefile을 만들어주는 옵션
```

`CMakeLists.txt`를 사용해서 cmake가 Generate
-> Makefile : make
-> binaries 생성

예시
```bash
$ cmake -S . -B build 
```

## 버전 출력
```bash
# CMakeLists.txt

message("cmake version is " ${CMAKE_VERSION})
```
`CMAKE_MAJOR_VERSION`
`CMAKE_MINOR_VERSION`

## 최소 버전
```bash
# CMakeLists.txt

cmake_minimum_required(VERSION 4.0) 
```

## 프로젝트 지정
```bash
# CMakeLists.txt

project({project_name})
```

## 실행 타겟
```bash
# CMakeLists.txt

project(helloworld)
add_executable(${CMAKE_PROJECT_NAME} hello.c)
```

```bash
$ cmake -B build # build dir
$ cmake --build build # build dir에 실행파일 생성
```
